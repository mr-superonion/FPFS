# FPFS: Fourier Power Function Shaplets (A fast, accurate shear estimator)

## Installation

For stable version:

pip install FPFS

Or clone the repository:

git clone https://github.com/mr-superonion/FPFS.git
cd FPFS
pip install -e .

## Demos

Isolated galaxies

demo1 simulates noisy galaxies with different SNRs and processes the galaxies with FPFS shear estimator. + fpfs.simutil.sim_test: a wrapper of galsim to simulate galaxies for simple tests; + fpfs.fpfsBase.fpfsTask: a task to process galaxy images and measure shear.

demo2 estimates shear under a more realistic situation with PSF errors.
